# QATextView

支持固定格式的文本高亮展示、支持输入自定义的emoji<br>
==========================================

主要功能点:<br>
------------
1. 支持自定义Emoji表情的输入<br>
2. 支持手动输入自定义的Emoji表情  (输入[微笑] ---> 😊)<br>
3. 支持url的高亮显示<br>
4. 支持话题(#Topic#)的高亮显示<br>
5. 支持@的高亮显示<br>
